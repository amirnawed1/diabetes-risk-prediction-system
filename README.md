# Diabetes Prediction Streamlit App

## Files to keep for deployment
- app.py
- train_model.py
- diabetes.csv
- diabetes_model.pkl
- requirements.txt

## How to create model file locally
Run:
```bash
python train_model.py
```

Then run app:
```bash
streamlit run app.py
```

## Streamlit Cloud
Upload all files to GitHub and deploy `app.py` from Streamlit Cloud.
